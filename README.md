# Criando um Dashboard de Vendas do Xbox com Excel

## Desafio Bootcamp DIO-Santander Analise de Dados 2025

Criar um Dahsboard em Excel com base em dados ficticios fornecidos pelo curso

Aqui a ideia foi criar um Dashboard que auxiliasse a gerência comercial a ter uma overview sobre o cenário atual das vendas de subscriptions, fornecendo detalhes como valores totais e valores individuais dos passes especiais.
A geração de pedidos mês a mês, com filtragem em tipo do plano e de renovação.

Adicionalmente um grafico que indica o vencimento dos planos para os próximos meses, de acordo com o tipo de plano do clinete(mensal, trimestral e anual), possibilitando uma noção de forecast e ações necessárias para buscar a renovação com os clientes.
